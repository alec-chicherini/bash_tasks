Задание: 
Чтобы ацки не загружать HDD скрипт дописывает 1-2 Mb(а не 1-2Gb).
logrotate создаёт одну копию лога как только размер превышает 10Мб.
Файлы:
script - основной скрипт который используется далее в задниях. Добавляет данные к файлу, и запускает logrotate.
script_init cleanup - Скрипты копируют/удаляют файлы. Добавляют/удаляют строку в crontab.

Использование:

#инициализация
cd task0
./script_init like cron

#проверка
ls -lah /var/log/time*

#очистка
./cleanup
